# Project: Bookmark Manager

## User Profile
- Non-coder / beginner
- Needs step-by-step explanations
- Prefers simple, barebones solutions

## Tech Stack
- Backend: Supabase (PostgreSQL + Auth + API)

## Data Model
### Bookmarks Table
- `id` - unique identifier (auto-generated)
- `title` - text (mandatory)
- `url` - text (mandatory)
- `notes` - text (optional)
- `user_id` - linked to authenticated user
- `created_at` - timestamp

## Preferences
- Keep everything simple and minimal
- Avoid unnecessary complexity
- Explain concepts in plain language
